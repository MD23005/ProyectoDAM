# Alquileres UES
Esta aplicación móvil permite registrar un catálogo de vehículos, clientes y alquileres, con el objetivo de brindar una herramienta de gestión para un negocio de alquiler de automóviles.

## Tecnologías utilizadas
- Java 
- XML
- SQLite

## Requisitos para correr el proyecto
- Android mínimo: 15 (API 35)
- Android objetivo: 16 (API 36)
